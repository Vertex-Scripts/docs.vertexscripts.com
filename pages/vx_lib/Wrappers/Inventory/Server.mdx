# Server

Supported inventories:
* ox_inventory
* qb-inventory
* qs-inventory

## addItem
Adds an item to the inventory.

`vx.inventory.addItem(source, item, count)`

* source: `string` or `number`
* item: `string`
* count?: `number`

## removeItem
Removed an item from the inventory.

`vx.inventory.removeItem(source, item, count)`

* source: `string` or `number`
* item: `string`
* count?: `number`

## getItemCount
Gets the count of an item in the inventory.

`vx.inventory.getItemCount(source, item)`

* source: `string` or `number`
* item: `string`

Returns: `number`

## hasItem
Checks if the inventory has an item.

`vx.inventory.hasItem(source, item, count)`

* source: `string` or `number`
* item: `string`
* count?: `number`

Returns: `boolean`
