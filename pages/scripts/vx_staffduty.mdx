# Staff duty
This script is ascript that allows you to check if a player is on duty.

## Installation
1. Put the `vx_staffduty` folder in your `resources` folder.
2. Add `ensure vx_staffduty` to your `server.cfg`.
3. Start the script

## Exports

### Client

```lua
function inDuty();

-- Example
RegisterCommand("broadcast", function()
  local isOnDuty = exports.vx_staffduty:inDuty()
  if not isOnDuty then
    print("You are not on duty")
    return
  end
end, true)
```

### Server

```lua
function isOnDuty(source: number);

-- Example
RegisterCommand("broadcast", function()
  local isOnDuty = exports.vx_staffduty:isOnDuty(source)
  if not isOnDuty then
    print("You are not on duty")
    return
  end
end, true)
```
